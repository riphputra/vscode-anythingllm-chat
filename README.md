# AnythingLLM Assistant

AI Chat and Agent Assistant powered by your own AnythingLLM instance.

## Features
- Chat with your AI directly in VS Code.
- Scans your entire project folder to provide context to the AI.
- Connects to your self-hosted AnythingLLM via API.
- Supports custom model routers configured in AnythingLLM.

## Requirements
- A running instance of AnythingLLM.
- A valid API Key from your AnythingLLM dashboard.
