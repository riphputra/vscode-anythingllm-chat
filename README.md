# 🎉 AnythingLLM Assistant v1.0.0

Rilis pertama dari ekstensi VS Code yang mengintegrasikan AnythingLLM langsung ke editor Anda!

## ✨ Fitur Utama
- 🤖 Chat dengan AI AnythingLLM langsung dari VS Code
- 🔍 Scan seluruh project untuk konteks AI yang lebih akurat
- 🔌 Koneksi ke self-hosted AnythingLLM via API
- 🎨 UI modern dengan sidebar dedicated

## 📥 Cara Install
1. Download file `.vsix` di bawah
2. Buka VS Code → Extensions (Ctrl+Shift+X)
3. Klik `...` → Install from VSIX...
4. Pilih file yang didownload

## ⚙️ Konfigurasi
Buka Settings (Ctrl+,) dan cari "AnythingLLM":
- **Server Url**: `http://localhost:3001`
- **API Key**: Dapatkan dari dashboard AnythingLLM
- **Workspace Slug**: Nama workspace Anda
